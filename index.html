A POS Data Visualisation on Clustering

This is an attempt to implement web charts with d3.js library, as a bonus assignment for my university course. The Point Of Sale data are only for educational reasons. The data were cleanced, tranformed and aggregated in order to be ready for data mining. Clustering was implemented with Rapid Miner and the results are plotted below:




Below you can find a Baskets per Weekday and Cluster chart. Using the selector, you can pick the cluster of your choice and on the chart to the right, the number of basket for each day of week will be shown. Notice that Sunday is missing, since the Supermarket is not operating that day.

Based on the charts, we can observe that Dairy and Personal Care are the most popular and lucrative clusters for the Supermarket. We can also notice that Diary baskets are decreased on Wednesday and Friday, which are fasting days for the Orthodox Greeks. In conclusion, using visualisations we can extract valuable information about some unstructured data, which can help us make strategic decisions.
